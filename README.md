## Overview

**PSIT Virtual Labs** is a comprehensive virtual laboratory platform developed for **Pranveer Singh Institute of Technology (PSIT)** students. It provides interactive simulations and structured learning materials across multiple engineering and computer science disciplines.

> **Live Platform:** [https://psit-virtual-labs.vercel.app/](https://psit-virtual-labs.vercel.app/)

---

## Key Features

| Feature | Description |
|---------|-------------|
| **Interactive Simulations** | Hands-on virtual experiments with real-time visualization |
| **7+ Subject Labs** | Comprehensive coverage of CS/Engineering subjects |
| **80+ Experiments** | Well-documented experiments with step-by-step guidance |
| **Structured Learning** | Theory → Pre-test → Simulation → Post-test → Feedback |
| **Responsive Design** | Works seamlessly on desktop, tablet, and mobile |
| **No Installation** | Browser-based, accessible anywhere |

## Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Edge, Safari)
- No additional software installation required

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/Psit-Virtual-Labs.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Psit-Virtual-Labs
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server:
   npx serve .
   # Or with Python:
   python -m http.server 8000
   ```

4. **Access the labs**
   ```
   http://localhost:8000
   ```

## Contributing

We welcome contributions! Here's how you can help:
